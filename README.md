# Delivery Operations Analysis using SQL

## Project Overview

This project analyzes a delivery operations dataset using SQL to identify factors affecting delivery performance, warehouse efficiency, and customer experience.

## Objectives

- Explore delivery operations data
- Identify factors contributing to delivery delays
- Analyze warehouse performance
- Investigate the relationship between weather and delivery times
- Generate business insights from operational data

## Dataset

The dataset contains information about:

- Orders
- Customers
- Cities
- Warehouses
- Product categories
- Shipping modes
- Weather conditions
- Delivery times
- Delivery delays

## SQL Concepts Used

- SELECT
- DISTINCT
- COUNT
- AVG
- GROUP BY
- Aggregate Functions

## Analysis Performed

1. Product category analysis
2. City-wise order analysis
3. Weather condition analysis
4. Delivery delay analysis
5. Warehouse performance analysis
6. Shipping mode comparison

## Key Insights

1. Total Product Categories = 6, electronics, clothing, books, home and kitchen, beauty
2. Total orders = 350
3. Clear Weather has highest amount of deliveries
4. Snowy Weather has highest average delay. 
5. Economy Shipping mode has highest average delay. 
6. City with Highest delay count
